# Beat the Odds 🎲

**Interactive trading simulation showing that winrate alone means nothing without risk:reward.**

[![Live Demo](https://img.shields.io/badge/demo-live-blue?style=flat-square)](https://beat-the-odds.pages.dev)
[![License](https://img.shields.io/badge/license-MIT-green?style=flat-square)](LICENSE)

## Overview

This simulation demonstrates what happens when **1000 chimpanzees each make 100 completely random trades**.  
Each trade has a **50% chance of winning**, without any strategy.  

The results reveal:  
- How many appear to be “skilled traders” purely by luck  
- The minimum **risk:reward ratio** required to actually be profitable
